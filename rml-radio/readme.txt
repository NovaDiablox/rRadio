# RML Radio Addon

## Description
The RML Radio addon is a Garry's Mod addon that allows players to listen to their favorite radio stations in-game. It provides a seamless integration with the game, allowing players to enjoy music while playing.

## Features
- Access to a wide range of radio stations from around the world.
- Simple and intuitive user interface for easy navigation and station selection.
- Customizable settings to personalize the radio experience.
- Support for both singleplayer and multiplayer modes.

## Installation
1. Download the RML Radio addon from the Steam Workshop or any other trusted source.
2. Extract the addon files to the `addons` folder in your Garry's Mod installation directory.
3. Launch Garry's Mod and enable the RML Radio addon in the Addons menu.

## Usage
1. Open the RML Radio menu by pressing the designated key (default: K).
2. Browse through the available radio stations using the arrow keys or mouse scroll.
3. Select a station by pressing the Enter key or left-clicking on it.
4. Adjust the volume and other settings as desired.
5. Enjoy listening to your favorite radio station while playing Garry's Mod!

## Configuration
The RML Radio addon provides several configuration options to customize your experience. These options can be accessed through the in-game settings menu. Here are some of the available configuration options:

- Keybindings: Change the key used to open the RML Radio menu.
- Volume: Adjust the volume level of the radio.
- Favorites: Save your favorite radio stations for quick access.
- Language: Change the language used in the addon interface.
